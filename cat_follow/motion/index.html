Movement Calibration

Use these tools to measure and set the car's movement parameters.

1. Run Physical Tests

Place the car on a flat surface with plenty of space.

Speed Test:

After running, measure the distance traveled in cm.

Steering Test (Left):

After running, measure the diameter of the circle in cm. Radius = Diameter / 2.

Steering Test (Right):

After running, measure the diameter of the circle in cm. Radius = Diameter / 2.

2. Enter and Save Values

Edit the JSON below with your measured values and click Save.
